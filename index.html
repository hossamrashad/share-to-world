charset / utf-8

<meta charset="utf-8>

Its usefulness is to specify the encoding of the page and here we use the universal encoding Unicode Transformation Format which supports all languages and their characters.


فائدته تحديد الترميز الخاص بالصفحة ونستخدم هنا الترميز العالمي Unicode Transformation Format والذي يدعم جميع اللغات والأحرف الخاصة بها.

author / web author

<meta name="author" content="Author Name">

this tag is used to name the author of the page


تستخدم هذه العلامة لتسمية مؤلف الصفحة.

revisit after

<meta name="revisit-after" content="10 days">

it gives a command to the robots to return to a page after a specific period of time.


يعطي الأمر للروبوتات للعودة إلى الصفحة بعد فترة زمنية محددة.

cache control

<meta http-equiv="cache-control" content="no-cache">

this tag is set in hopes of controlling when and how often a page is cached in the browser.

it's best to do this in this in the HTTP header.


تم تعيين هذه العلامة على أمل التحكم في وقت وعدد مرات تخزين الصفحة في ذاكرة التخزين المؤقت في المتصفح.

من الأفضل القيام بذلك في رأس HTTP.

http-equiv refresh

<meta http-equiv="refresh" content="60">

refresh is a method of constructing a web browser to automatically refresh the current web page or frame after a given time interval, using an HTML meta element.


التحديث هو طريقة لإنشاء متصفح ويب لتحديث صفحة الويب الحالية أو الإطار تلقائيًا بعد فترة زمنية معينة ، باستخدام عنصر HTML meta.

viewport

<meta name="viewport" content="width=device-width,  initial-scale=1, shrink-to-fit=no">

This gives the browser instructions on how to control the page's dimensions and scaling.


يوفر هذا إرشادات للمتصفح حول كيفية التحكم في أبعاد الصفحة وقياسها.

description

<meta name="description" content="Your Website Description">

Its usefulness is to put the page description when you search for a specific site in Google, for example, you find the name of the site and under it a text describing the site and this is the Meta Description.


فائدته وضع وصف الصفحة فعندما تبحث عن موقع معين في Google مثلا تجد إسم الموقع وتحته نص يوصف الموقع وهذا هو ال Meta Description.

copyright

<meta name="copyright" content="Your Company Or Brand Name">

Its usefulness is to tell the person browsing your site that you have material that has intellectual rights and is owned by your company or your site written within the content.


فائدتها أن تخبر الشخص الذي يتصفح موقعك أنه لديك مواد عليها حقوق فكرية Copyright وهي مملوكة لشركتك أو موقعك المكتوب داخل ال Content.

robots

<meta name="robots" content="index, follow">

With it, you can direct the search spiders of search engines to what you want. You can specify the pages you want to index and the pages you do not want and have complete control over what happens.


عن طريقه يمكنك توجيه عناكب البحث Search Spiders الخاصة بمحركات البحث لما تريد فيمكنك تحديد الصفحات التي تريد أن تقوم بعمل Indexing لها والصفحات التي لا تريد وتتحكم كاملا فيما يحدث.

revised

<meta name="revised" content="Tuesday, July 20th, 2021, 10:15 am">

It is the page review section. You can put the date you reviewed the page. For example, you wrote a specific article. You can put this section to show that you recently updated the article to appear in the person’s search engine before entering the article.


هو الجزء الخاص بمراجعة الصفحة فيمكنك وضع التاريخ الذي قمت بمراجعة الصفحة فيه وعلى سبيل المثال قمت بكتابة مقال معين يمكن أن تضع هذا الجزء لتوضح انك قمت بتحديث المقال مؤخرا ليظهر في محرك البحث للشخص قبل الدخول على المقال.

IE=edge

<meta http-equiv="X-UA-Compatible" content="IE=edge">

The need to generate CSS files to specifically handle older IE versions is something many old web developers can remember.


إن الحاجة إلى إنشاء ملفات CSS لمعالجة إصدارات IE الأقدم على وجه التحديد أمر يمكن للعديد من مطوري الويب القدامى تذكره.

keywords

<meta name="keywords" content="Information of the page.">

It can be used to give more information to search engines about the topic of the page.


يمكن استخدامه لإعطاء مزيد من المعلومات لمحركات البحث حول موضوع الصفحة.
